How to install LumiROM

Luminous | 418 • December 23, 2025

If you already got TWRP prepared, you now can install LumiROM. This guide will be for the installation of it.

1. Recovery Update

  • If you already have some kind of recovery, you will need to update to get erofs support and avoid mount errors.
  • This works on all ROMs, so there is no issue. Please download it here: TWRP (EROFS Support).

2. Download the ROM

  • Go to this link to find the downloads.
  • You will need 2 files: LumiROM and LumiVENDOR.

3. Transfer & Installation

After downloading the files, you can choose one of two methods:

  • Option A: Internal Storage - Move the files directly to your device storage and locate them in TWRP to install.
  • Option B: ADB Sideload - Use a PC to flash the files.
Sideload Setup: You need platform-tools (ADB commands).
You can use this script to automate the setup: Click to download Script.
  • To Sideload: In TWRP, go to Advanced > ADB Sideload.
  • On PC, type: adb sideload <path_to_file>

4. Final Steps

  • I first recommend to install LumiROM, and then install LumiVENDOR for everything to work correctly.
  • Format Data: Required if coming from Stock or another ROM. (Not required if updating from a previous LumiROM version).
  • If you did all things fine, the device should boot fine.
For any doubts, join the community: @lumiromgroup
